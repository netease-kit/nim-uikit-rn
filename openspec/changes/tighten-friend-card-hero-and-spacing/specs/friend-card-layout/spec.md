## ADDED Requirements

### Requirement: Friend card page must use compact hero and spacing

The RN friend card page MUST display the header nickname on a single truncated line and use a tighter overall layout density.

#### Scenario: Viewing the friend card page

- **WHEN** the user opens the RN friend card page
- **THEN** the header nickname shows at most one line with ellipsis
- **AND** information values such as alias, birthday, mobile, email, and signature show at most one line with ellipsis
- **AND** the page spacing is tighter than before
- **AND** the existing friend-card actions remain available
