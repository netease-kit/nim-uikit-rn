## ADDED Requirements

### Requirement: Conversation list rows must use a tighter visual scale

The RN conversation list MUST present avatars and row text at a tighter visual scale than the previous layout.

#### Scenario: Viewing the conversation list

- **WHEN** the user views conversation rows on the message tab
- **THEN** the avatar is smaller than before
- **AND** the title and subtitle text are smaller than before
- **AND** row interaction behavior remains unchanged
