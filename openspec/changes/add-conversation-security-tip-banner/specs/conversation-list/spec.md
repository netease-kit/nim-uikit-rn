## MODIFIED Requirements

### Requirement: Conversation List Security Tip Banner

The app SHALL display the anti-fraud security tip banner on the messages page header area.

#### Scenario: Security tip placement

- **WHEN** the user opens the conversation list page
- **THEN** the app MUST display a yellow anti-fraud tip banner below the search entry
- **AND** the banner MUST appear above the offline network banner and the conversation list content
