# Proposal

## Why

The RN conversation list currently uses avatars and title text that are slightly larger than desired for a compact mobile list.

## What Changes

- reduce the avatar size used by RN conversation rows
- reduce the title and subtitle sizing used by RN conversation rows
- keep conversation list behavior unchanged

## Impact

- affected spec: `conversation-list-row-density`
- affected code: `src/NEUIKit/rn/components.tsx`
