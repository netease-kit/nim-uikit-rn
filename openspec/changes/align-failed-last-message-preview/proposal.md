## Why

会话模块测试用例 `0260-会话列表外露消息显示与聊天详情页最后一条显示一致` 要求当聊天页最后一条消息发送失败时，会话列表预览显示 `【提醒消息】`。当前 React Native 会话预览会直接沿用失败消息原文，不满足该用例要求。

## What Changes

- 在会话预览同步逻辑中识别“最后一条消息发送失败”的场景。
- 将该场景的会话列表预览统一映射为 `【提醒消息】`。
- 保持删除最后一条消息回退上一条、撤回显示 `此消息已撤回` 的现有逻辑不变。

### Modified Capabilities

## Impact

- 受影响代码：`stores/MessageStore.ts`
- 受影响行为：会话列表对发送失败最后一条消息的预览展示
- 无新增依赖，无接口协议变更
