# 重复收到相同好友验证消息

## 基本信息

- 工作表: 【易用性】产品项目
- 用例序号: 0119
- 用例分级: P0
- 用例Suite: 【易用性】产品项目 > IM UIkit > IM UIKit-RN > 10.0.0 > 通讯录
- 用例Tag: Android端已自动化
- 用例别名: NIM-265619

## 前置条件

无

## 执行步骤

1.收到10条相同验证消息A 2.阅读验证消息后回到通讯录再次收到1条相同验证消息A 3.拒绝验证消息A后再次收到相同验证消息

## 预期结果

1.通讯录验证消息未读数数量展示1，验证中心验证消息A仅展示一条，验证消息头像上展示该条验证消息数量10且展示新验证消息底色，返回通讯录页面后所有未读数清空2.通讯录验证消息数展示1，进入验证中心相同验证消息A上不展示未读数，展示新的底色且排在其他验证消息上方3.通讯录验证消息未读数数量展示1，验证中心验证消息A仅展示一条，且不覆盖已拒绝的验证消息
