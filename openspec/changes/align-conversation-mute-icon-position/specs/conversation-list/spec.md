## MODIFIED Requirements

### Requirement: Conversation List Meta Layout

The app SHALL align the conversation-row right-side meta area with the product layout for muted conversations.

#### Scenario: Muted conversation row

- **WHEN** a conversation row is muted
- **THEN** the mute icon MUST render below the conversation time
- **AND** the mute icon MUST be right-aligned within the meta area
- **AND** the mute icon size MUST match the subtitle text font size
