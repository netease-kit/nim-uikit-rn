## ADDED Requirements

### Requirement: Profile-card information rows must use a compact single-line layout

The RN profile-card pages MUST use single-line information rows with a compact height suited for profile metadata.

#### Scenario: Viewing profile-card information rows

- **WHEN** the user views information rows on friend-card or my-detail pages
- **THEN** each single-line information row uses the compact profile-card row height
- **AND** information values continue to truncate with ellipsis when needed
