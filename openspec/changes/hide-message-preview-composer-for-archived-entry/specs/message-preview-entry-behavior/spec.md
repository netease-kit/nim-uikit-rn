## ADDED Requirements

### Requirement: Archived message preview entries must hide the composer

The RN message preview page MUST hide the bottom composer when the page is opened from pinned messages or collections.

#### Scenario: Opening message preview from pinned messages or collections

- **WHEN** the user opens the message preview page from the pinned-message list or collection list
- **THEN** the page displays the message content without the bottom composer
- **AND** other message preview entry paths continue to show the composer
