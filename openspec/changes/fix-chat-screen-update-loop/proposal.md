# Change: 修复聊天页循环更新报错

## Why

聊天详情页当前在真机运行时会持续报出 `Maximum update depth exceeded`。该问题会导致页面重复渲染、键盘联动不稳定，并干扰输入框与底部工具栏的正常显示。

## What Changes

- 收敛聊天页中会导致导航头部或页面 effect 重复触发的不稳定引用。
- 保持聊天页标题、在线状态和消息列表相关状态在相同输入下输出稳定。
- 确保聊天页不会因为同一轮渲染中的重复状态写入而陷入更新循环。

## Impact

- 影响聊天详情页的头部标题、副标题、在线状态展示与页面渲染稳定性。
- 不改变聊天功能协议和已调好的 Android 键盘抬升参数。
