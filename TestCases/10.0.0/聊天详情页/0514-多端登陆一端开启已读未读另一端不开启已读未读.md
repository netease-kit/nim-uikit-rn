# 多端登陆一端开启已读未读另一端不开启已读未读

## 基本信息

- 工作表: 【易用性】产品项目
- 用例序号: 0514
- 用例分级: P1
- 用例Suite: 【易用性】产品项目 > IM UIkit > IM UIKit-RN > 10.0.0 > 聊天详情页
- 用例Tag: 无
- 用例别名: NIM-265619

## 前置条件

无

## 执行步骤

1.A端开启已读未读功能，B端不开启已读未读功能，B端发送消息1，A端发送消息2
2.B端开启已读未读，查看消息1和2

## 预期结果

1.在A端消息1不显示已读未读标志
在B端消息1不显示已读未读标志，
在A端消息2显示已读未读标志
在B端消息2不显示已读未读标志2.在B端消息1不展示已读未读，消息2展示已读未读

## 备注

1、iOS是开启方都显示已读未读，未开启的都不显示
目前已读未读是本地设置，不是进行多端的同步
