# 系统权限请求弹窗-允许部分

## 基本信息

- 工作表: 【易用性】产品项目
- 用例序号: 0854
- 用例分级: P0
- 用例Suite: 【易用性】产品项目 > IM UIkit > IM UIKit-RN > 10.0.0 > 相册权限
- 用例Tag: 无
- 用例别名: NIM-265618

## 前置条件

1.展示相册权限请求弹窗

## 执行步骤

1. 在权限弹窗中点击 "允许部分照片访问"
2. 在系统照片选择界面勾选 3 张照片
3. 点击确认按钮

## 预期结果

仅能查看和选择已勾选的 3 张照片，未勾选照片不可见

## 备注

可在设置中再次添加更多照片
发送时能看到新增加的图片
