## MODIFIED Requirements

### Requirement: Media viewer bottom action icons use Android-aligned sizing and contrast

RN image and video viewers MUST use bottom action icon sizing and contrast aligned with the Android implementation.

#### Scenario: Viewer bottom actions are rendered

- **WHEN** the image or video viewer renders its bottom action area
- **THEN** the action icons use compact icon-sized touch targets instead of oversized circular buttons
- **AND** the spacing and bottom margins align with the Android visual scale
- **AND** the buttons remain visually distinguishable on light-colored image or video content
