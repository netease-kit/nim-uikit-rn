# Feature Support Matrix

判定口径：

- 以当前项目的**实际功能露出**为准
- **不明确一律按不支持处理**
- `已支持` 表示当前 App 有明确入口且可完成主要操作闭环
- `未支持` 表示当前无明确入口，或无法明确确认该功能已经完整露出

主要参考页面：

- `app/(tabs)/index.tsx`
- `app/(tabs)/contacts.tsx`
- `app/chat/[id].tsx`
- `app/session/p2p-settings.tsx`
- `app/team/settings.tsx`
- `app/user/setting.tsx`
