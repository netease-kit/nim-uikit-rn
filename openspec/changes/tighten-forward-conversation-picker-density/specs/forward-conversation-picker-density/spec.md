## ADDED Requirements

### Requirement: Forward conversation picker must use compact list density

The RN forward conversation picker MUST present recent targets and conversation targets with compact spacing appropriate for mobile forwarding flows.

#### Scenario: Browsing forward targets

- **WHEN** the user opens the RN forward conversation picker for serial or merged forwarding
- **THEN** the page uses tightened vertical spacing between sections
- **AND** recent targets use compact avatar and label spacing
- **AND** each forward target row uses a compact height and text spacing without changing selection behavior
