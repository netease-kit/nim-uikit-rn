# IM Feature Map

用于从零定义 IM 产品能力范围。

## MVP 功能集

### 鉴权
- 验证码登录或账号登录
- 登录态持久化
- 登出
- 会话恢复

### 会话
- 会话列表
- 未读数
- 会话预览
- 进入会话

### 聊天
- 文本消息
- 历史消息
- 发送态
- 重试

## 标准功能集

在 MVP 基础上增加：
- 会话置顶
- 免打扰
- 消息撤回
- 转发
- 已读回执
- 图片 / 文件消息
- 好友申请
- 好友备注
- 建群 / 邀请 / 退群
- 通知设置

## 完整功能集

在标准功能集基础上增加：
- 合并转发
- pin
- 收藏
- 举报 / 反垃圾
- 聊天记录检索
- 群管理
- 黑名单
- 多端 / 被踢处理

## 选型规则

- 用户未指定范围时，优先先给 MVP
- 用户明确要“完整 IM”，再扩展到标准或完整功能集
