# Relationship And Team Extensions

用于从零规划好友扩展和群组管理能力。

## 覆盖范围

- 好友申请
- 好友备注
- 黑名单
- 群资料编辑
- 群成员管理
- 群禁言
- 退群 / 解散群

## 模块契约

### 页面层

- 联系人首页
- 好友申请页
- 好友资料页
- 备注编辑页
- 群设置页
- 群成员页

### 状态层

- 好友列表、申请列表、黑名单
- 群资料、群成员、我的群角色

### 服务层

- addFriend / accept / reject / updateRemark / blockUser
- createTeam / inviteMembers / updateTeamInfo / kick / leave / dismiss

## 功能拆解

### 好友申请

最少测试：

- 发起好友申请
- 通过申请
- 拒绝申请
- 未读申请数变化

### 好友备注

核心要求：

- 好友资料页可以进入备注编辑
- 保存后联系人列表、会话显示名同步更新

最少测试：

- 正常保存备注
- 清空备注
- 超长备注
- 刷新后备注仍正确
- 会话显示名联动

### 黑名单

最少测试：

- 添加黑名单
- 移除黑名单
- 黑名单列表刷新

### 群资料编辑

最少测试：

- 修改群名称
- 修改群介绍
- 修改群头像
- 修改我的群昵称

### 群成员管理

最少测试：

- 邀请成员
- 踢人
- 成员列表刷新
- 成员角色限制正确

### 群禁言 / 退群 / 解散群

最少测试：

- 群禁言开关
- 普通成员退群
- 群主解散群
- 成功后列表与详情联动

## 验收标准

- 好友和群组模块都具备独立闭环
- 页面、列表、详情、会话昵称联动正确
- 管理类动作受角色和权限约束
