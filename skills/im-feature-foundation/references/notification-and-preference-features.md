# Notification And Preference Features

用于从零规划通知、消息提醒和个人偏好能力。

## 覆盖范围

- 全局通知开关
- 通知详情展示开关
- 声音 / 振动
- 已读回执开关
- 外观切换
- 权限状态查询和跳转系统设置

## 模块契约

### 页面层
- 设置首页
- 通知设置页
- 外观页

### 状态层
- PreferenceStore 持有偏好值和权限状态

### 服务 / 平台层
- 通知权限查询
- 通知权限申请
- 打开系统设置

### 工具层
- 本地存储
- 平台差异兼容

## 功能拆解

### 通知总开关

最少测试：
- 开启通知
- 关闭通知
- 保存后重进仍正确

### 详情显示 / 声音 / 振动

最少测试：
- 分项开关生效
- 保存后重进仍正确
- 推送内容是否按配置展示

### 已读回执开关

最少测试：
- 开启后发送时带回执配置
- 关闭后不带回执配置

### 外观切换

最少测试：
- 外观设置变化
- 页面重进后保持一致

### 权限管理

最少测试：
- 获取权限状态
- 请求权限
- 权限拒绝后提示用户前往系统设置

## 验收标准

- 偏好值有统一状态源
- 所有开关具备持久化
- 平台不支持的能力有降级处理
