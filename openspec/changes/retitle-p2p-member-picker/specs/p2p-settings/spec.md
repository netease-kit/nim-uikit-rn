## MODIFIED Requirements

### Requirement: P2P Add-Member Picker Presentation

The app SHALL present a neutral selection page when the user enters the add-member flow from p2p settings.

#### Scenario: Open picker from p2p settings header add icon

- **WHEN** the user taps the add-member icon in the p2p settings header area
- **THEN** the destination page title MUST be `选择`
- **AND** the page MUST NOT show the helper module that explains current p2p members will join automatically
