# 在线离线测试用例执行记录

## 执行范围

- 用例目录：`TestCases/10.0.0/在线离线`
- 本轮用例：本地会话/云端会话列表单聊对象上线离线、好友列表好友上线离线、单聊页面对方上线离线。
- 执行方式：按 `AGENTS.md` 要求一次只验证一条用例。当前仓库没有自动化 runner，且在线离线终态依赖双账号和 SDK 状态推送，因此采用“读取需求 -> 核对实现路径 -> 修复缺失 UI/订阅链路 -> 静态校验/启动验证 -> 记录结果”。真实上线/离线翻转需双端账号复测。
