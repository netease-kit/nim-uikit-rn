## ADDED Requirements

### Requirement: Recent Forward Targets Fit Five Per Row

The forwarding target selection page SHALL size the recent-forward target items so five targets fit in one visible row on standard phone widths, using reduced spacing and smaller avatars where necessary.

#### Scenario: Recent forward targets on phone width

- **WHEN** the forwarding target selection page shows the recent-forward section on a standard phone-width viewport
- **THEN** one visible row MUST fit five recent-forward targets without requiring horizontal scrolling to see the fifth item
- **AND** the recent-forward target avatars MUST remain visually balanced with their labels
